# How to Add Your Welcome Banner Image

To add a custom welcome banner to your README:

1. **Create or find a banner image** (recommended size: 1200x300 pixels)
2. **Save it in your repository** (e.g., in the root or an `assets` folder)
3. **Update the README.md** - Replace this line:
   ```
   ![Welcome Banner](path/to/banner.png)
   ```
   With your image path, for example:
   ```
   ![Welcome Banner](./assets/welcome-banner.png)
   ```
   Or if using a URL:
   ```
   ![Welcome Banner](https://your-image-url.com/banner.png)
   ```

**Banner Suggestions:**
- Use a dark theme to match your README
- Include minimal text or graphics
- Keep it clean and professional
- Recommended formats: PNG, JPG, or SVG

**Free Banner Tools:**
- Canva (banner templates)
- Figma
- Adobe Express
- GitHub Readme Banner Generator (online tools)
